## The Setup Process: Easier Than You'd Think

One thing that surprised me while researching this: setup is actually pretty straightforward. You don't need special tools or engineering degrees.

For the Capsule/Seed24:
1. Write down your seed phrase (you already have this from your hardware wallet)
2. Slide the letter tiles onto the core in order
3. Use separators between words
4. Lock it with the cotter pin
5. Screw on the cap
6. Apply tamper-evident seal (on Seed24)
7. Hide it somewhere safe

The Cassette works similarly but with a different form factor - tiles slot into a credit-card-sized frame.

The only real "gotcha" is making sure you don't lose tiles during setup. They're small, they can roll, and if your cat decides to help, you might be buying a replacement set. Work over a tray or on a clean desk.
